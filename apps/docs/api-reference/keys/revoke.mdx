---
title: "Revoke Key"
description: "Delete an api key for your users"
api: "DELETE /v1/keys/:keyId"
authMethod: "bearer"

---

## Request

<ParamField path="keyId" type="string" required>
The ID of the key you want to revoke.
</ParamField>


## Response

`202 OK`

<RequestExample>



```sh
curl --request DELETE \
  --url https://api.unkey.dev/v1/keys/key_123 \
  --header 'Authorization: Bearer <UNKEY>'
```





</RequestExample>

<ResponseExample>
```
OK
```


</ResponseExample>
