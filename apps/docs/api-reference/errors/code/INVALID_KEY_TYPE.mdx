title: KEY_USAGE_EXCEEDED
---

```json JSON response
{
  "error": {
    "code": "KEY_USAGE_EXCEEDED",
    "message": "what happened"
    ...
  }
}
```

## Problem

The key has been used too many times and is no longer valid.

## Solution

Update the `remaining` verifications on the key or create a new one.

If that doesn't help,Please get in touch on [Discord](https://unkey.dev/discord)
