## limit

### Example Usage

```python
import os
from unkey import Unkey

s = Unkey(
    bearer_auth=os.getenv("UNKEY_BEARER_AUTH", ""),
)

res = s.ratelimits.limit(request={
    "identifier": "user_123",
    "limit": 10,
    "duration": 60000,
    "namespace": "email.outbound",
    "cost": 2,
    "resources": [
        {
            "type": "organization",
            "id": "org_123",
            "name": "unkey",
        },
    ],
})

if res.object is not None:
    # handle response
    pass

```
