---
title: "Delete A Role"
description: "Delete a role from the Unkey system"
---

<RequestExample>

```ts
const { result, error } = await unkey.permissions.deleteRole({
  roleId: "role_45TTRKT5Yh28M6Qj",
});

if (error) {
  // handle potential network or bad request error
  // a link to our docs will be in the `error.docs` field
  console.error(error.message);
  return;
}
console.log(result);
```

</RequestExample>

<ResponseExample>
```json
{
  "result": {}
}
```
</ResponseExample>


<Note>
To use this function, your root key must have the `rbac.*.delete_role`
</Note>

## Request

<ParamField body="roleId" type="string" required>
    The id of the role to delete

    Example: `"role_123"`
</ParamField>

## Response
<ResponseField name="result">
    `{}`
</ResponseField>
