---
title: Audit Logs
description: 'Audit logs for your workspace, allowing you to see the history of all the resource requests made inside your workspace.'
---

The Audit Logs feature allows you to see the history of all the resource requests made inside your workspace. You can see the history of all the requests made to your workspace, including the API creation, API deletion, API key creation, API key deletion, and API key regeneration. To gain access please reach out to Unkey support or join our Discord.

## Accessing Audit Logs

To access the Audit Logs, click on the **Audit Logs** option in the left navigation bar.
