description: Overview of GitHub integrations in Unkey
---

Unkey has two distinct GitHub integrations that serve different purposes:

1. **GitHub App for Repository Connections** - Allows users to connect their GitHub repositories to Unkey projects for automatic deployments
2. **GitHub Secret Scanning** - Partnership program that detects leaked Unkey API keys in public repositories
