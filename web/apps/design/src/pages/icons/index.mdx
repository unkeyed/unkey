## Import

Every icon is a named export of the `@unkey/icons` package. Import the ones you
need individually.

```tsx
import { ArrowRight, Key, Trash } from "@unkey/icons";

<Key iconSize="xl-regular" />
```

## Gallery

Click a cell to copy its JSX tag. Icons are rendered at their default size
(`xl-thin`). Pass an `iconSize` prop to scale them up or down from there.
