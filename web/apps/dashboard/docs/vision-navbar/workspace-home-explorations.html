Workspace home — layout explorations

Four radically different arrangements of the workspace root page. All share: projects dominant, features as supplementary aggregate cards (Keyspaces, Ratelimit, Authorization, Identities, Logs). Each variant disagrees on how features are arranged and how projects are shown. Inspired by the Apr 21 call with Andreas — Vercel is a reference but these diverge from it.

V1 — Bento overview

projects + right-rail features Airy. Good for 3–15 projects.
