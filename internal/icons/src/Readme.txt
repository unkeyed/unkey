Here's the list of props you should pass to your icon instance:
- width: icon width (available if 'Use props for height/width values' was checked while exporting);
- height: icon height (available if 'Use props for height/width values' was checked while exporting);
- fill: icon color (available if 'Use props for inline colors' was checked while exporting);
- secondaryfill: icon secondary color (available if 'Use props for inline colors' was checked while exporting);
- strokewidth: icon stroke width (available if 'Use props for stroke-width values' was checked while exporting);
- title: icon title (available if 'Remove <title>' was not checked while exporting).
