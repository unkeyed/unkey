</Warning>

Each deployment's overview page includes real-time charts for throughput and latency. Navigate to a deployment from your project's **Deployments** tab to view them.

## Requests per second

The RPS chart shows the number of requests your deployment handles over time. Use it to spot traffic spikes, confirm scaling behavior, or correlate with incidents.

## Latency

The latency chart displays response time distributions over time. You can switch between p50, p75, p90, p95, and p99 percentiles. Check the [Requests](/observability/requests) tab to find individual slow requests and inspect their latency breakdown.

## Planned metrics

Unkey is adding per-deployment metrics for:

- CPU utilization
- Memory usage
- Instance count
- Disk usage
