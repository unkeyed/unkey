Unkey uses Stripe for all billing. Each workspace has its own subscription, payment method, and invoices. You manage billing from **Settings > Billing** in the dashboard.

## Plans

Unkey separates billing into two products:

- **Compute**: running and scaling deployed apps
- **API Management**: key verifications, rate limits, and API platform usage

Every workspace starts on the Free plan for API Management. Paid API Management plans unlock higher [limits](/platform/workspaces/quotas), longer data retention, and [team collaboration](/platform/workspaces/team-members). Compute plans include usage credits equal to the plan fee. Usage beyond the included credits is billed on top.

| Capability              | Free plan      | Paid plans                  |
| ----------------------- | -------------- | --------------------------- |
| API requests per month  | 150,000        | Varies by plan              |
| Log retention           | 7 days         | Varies by plan              |
| Audit log retention     | 30 days        | Varies by plan              |
| Team members            | Not available  | Included                    |
| Compute usage credits   | Not included   | Varies by plan              |

## Add a payment method

Before upgrading, you must add a payment method.

1. Navigate to **Settings > Billing**.
2. Click **Add payment method**.
3. Complete the Stripe checkout form with your card details and billing address.
4. After saving, you return to the billing page where you can select a plan.

## Upgrade your plan

1. Navigate to **Settings > Billing**.
2. On the **Compute** or **API Management** card, click **Choose a plan**, **Upgrade**, or **Change plan**.
3. Select a plan from the available options. Compute plans show included usage credits. API Management plans show the included request quota.
4. Confirm the selection.

Compute plan changes take effect immediately. Upgrades charge the difference now and add the difference as usage credits for the current period. Downgrades keep the current period's credits, and the lower fee starts next period.

API Management upgrades update your request quota immediately. API Management downgrades or cancellations take effect at the end of the current billing period.

## Downgrade or cancel

1. Navigate to **Settings > Billing**.
2. On the **Compute** or **API Management** card, click **Change plan** to downgrade or **Cancel plan** to cancel.
3. Confirm the cancellation.

Compute cancellation stops Compute immediately. Your deployments stop, no further Compute usage is billed, and usage up to cancellation is still charged.

API Management cancellation takes effect at the end of your current billing period. Your workspace continues to operate with paid plan limits until that date. After the period ends, the workspace reverts to Free plan [defaults](/platform/workspaces/quotas#free-plan-defaults).

If you cancel by mistake, you can resume your subscription from the same billing page before the period ends.

Once subscribed, the billing page shows your current Compute plan, API Management plan, usage, and options to change your plan, open the Stripe billing portal, or cancel.

## Billing portal

The billing portal lets you manage payment methods, view invoices, and update billing details through Stripe.

1. Navigate to **Settings > Billing**.
2. Click **Open Portal**.

The portal opens in a new page hosted by Stripe. From there you can:

- Update or replace your payment method
- Download past invoices
- Update billing contact information

## Usage tracking

The billing page displays usage for each product.

API Management usage combines key verifications and rate limit operations for the current billing month. The billing page and sidebar display this usage as a percentage of your monthly request quota.

Compute usage tracks CPU time, memory allocation over time, public egress, and reserved disk over time. The Compute card shows usage for the current period against the credits included in your plan.

Unkey sends email notifications when your workspace approaches or exceeds its quota.
