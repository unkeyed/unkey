---
title: "GitHub Secret Scanning"
description: "How Unkey protects you from leaked root keys"
---

Unkey has partnered with GitHub to scan repositories for leaked keys.

GitHub Secret Scanning uses regular expressions to scan repositories for keys matching Unkey root keys. If GitHub finds a key, it notifies Unkey. We validate the key and notify you via email. To keep your production environment running, we do not disable the key.

This works automatically — you don't need to set anything up. However, outside of GitHub, we can't detect leaked keys.

Learn more: [GitHub Secret Scanning](https://docs.github.com/en/code-security/secret-scanning)
