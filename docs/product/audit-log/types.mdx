## Workspaces

Workspace actions that will create a new audit log item.

- `workspace.create` - Creating a workspace.
- `workspace.update` - Updating a workspace.
- `workspace.delete` - Deleting a workspace.
- `workspace.opt_in` - Opting a workspace into a feature.

- `api.delete` - Deleting an API.

- `key.reroll` - Rerolling a key.
