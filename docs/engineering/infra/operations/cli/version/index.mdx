---
title: "Version"
description: "Manage API versions"
---

Create, list, and manage versions of your API. Versions are immutable snapshots of code, configuration, and infrastructure settings.

## Available commands

- `get`
- `list`
- `rollback`

## Command syntax

```bash
unkey version
```

## Quick reference

- `unkey version get`
- `unkey version list`
- `unkey version rollback`
