---
title: "Run"
description: "Run Unkey services"
---

Run Unkey services as standalone processes.

## Available services

- `api`
- `ctrl`
- `frontline`
- `krane`

## Command syntax

```bash
unkey run <service> [flags]
```

## Quick reference

- `unkey run api`
- `unkey run ctrl`
- `unkey run frontline`
- `unkey run krane`
